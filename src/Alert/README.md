## Installation

```bash
$ yarn add @uprise/Alert 
# or
$ npm i @uprise/Alert --save
```

## Quick Start

This will render an Alert notification:

```javascript
import { Alert } from '@uprise/alert'
```
```javascript
return <Alert type="primary">Hello world</Alert>
```
